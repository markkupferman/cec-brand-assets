# Chuck E. Cheese Brand Assets

Public bundle of brand-safe visual assets for Chuck E. Cheese marketing materials —
**logos, photography, character cutouts, brand backgrounds, and graphic elements.** Pull
any of these into a deck, a web page, or other materials by name or by stable
GitHub URL:

```
https://raw.githubusercontent.com/markkupferman/cec-brand-assets/main/<path-to-file>
```

All filenames are descriptive kebab-case so you can pick by intent without opening the file
first (e.g. `photos/food/pepperoni-pizza-cheese-pull.jpg`, `logos/chuck-e-cheese/cec-logo-horizontal-1-line-reversed.png`).

> ## ⚠️ Character integrity (BRAND RULE)
>
> **Never describe CEC characters visually in generated content** — not in filenames, not in
> alt text, not in deck narration. Reference them **by name + action only**, e.g.
> "Chuck E. waving," "Munch on keyboard," "Pasqually with pizza peel." This rule applies to
> Chuck E. Cheese, Jasper T. Jowls, Helen Henny, Mr. Munch, Chef Pasqually, Bella, and Ronnie
> in any form (illustrated, costumed, or 3D-rendered).
>
> When you need an image of one of these characters, pull from `characters/` directly rather
> than generating a description.

## 2. `characters/` — Character cutouts (PNG with transparency)

with multiple poses. Filename codes like `21cec135-...` (lowercased from the original campaign
shoot codes) mark the original-art assets; clean descriptive filenames (e.g.
`bella-jumping.png`) are the cleaned library copies. A `-bright` suffix means the
high-saturation recolored version — use when the background is dark or busy and the
character needs to pop. A `-silhouette` (or `-silhoutte`, misspelled in some legacy files)
is the solid-color silhouette. A `-cutout` file is the trimmed primary hero pose.

## 4. `logos/` — Logo lockups (PNG / JPG)

62 logo files across 10 sub-brand folders. All filenames are descriptive kebab-case so an AI
can pick by intent (layout + tagline + color treatment).

**Master CEC brand naming convention** (sub-brand folders follow simpler patterns):

- **Layout suffix:**
  - `horizontal-1-line` — mark + wordmark on one line (compact horizontal)
  - `horizontal-2-line` — mark on left + wordmark on two lines on right (taller horizontal)
  - `vertical` — mark above wordmark, centered (portrait/stacked composition)
  - `wordmark-horizontal` — wordmark only, one line (no mark)
  - `wordmark-stacked` — wordmark only, two lines (no mark)
  - `mark-roundel` — the circular character roundel by itself (no wordmark)
- **Tagline:** insert `-wakcbak` for "Where A Kid Can Be A Kid" tagline lockup.
- **Color treatment suffix:**
  - *(no suffix)* — full-color brand version
  - `-black` — solid black (monochrome print)
  - `-white` — solid white (use on dark/photo backgrounds)
  - `-reversed` — white wordmark + full-color mark (dark backgrounds while keeping the character roundel in brand red)
  - `-reserved` *(Mark-Roundel only)* — alternate transparent-edge cut of the full-color roundel (legacy brand-team label).
